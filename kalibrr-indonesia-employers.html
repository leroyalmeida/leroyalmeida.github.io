Indonesia Employer Page

CHALLENGE

Grow the number of Indonesian businesses/employers on the platform

DESIGN PROCESS

Duration: 3 weeks

Research: User feedback, stakeholder interview

Prototyping: Sketch mockups + Invision prototype

Testing: Iteration w/ stakeholders

Strategy: Product + marketing strategy

Visuals: Consistent with platform's visual design strategy

RESULTS

Successful launch

I worked with the Indonesia group to create a new landing page to cater for their growing number of clients in the country.

User interviews indicated that employers value three things:

  • A strong database of candidates
  • The online assessments feature
  • Great client/customer support

We showcased these features and added supporting statements from well-known client partners to emphasize these the value derived by these clients from the said value propositions.
